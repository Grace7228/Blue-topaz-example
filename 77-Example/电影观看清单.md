> 实现方法参考[[如何在Obsidian中添加电影卡片]]

```dataview
table without id ("![](" + cover + ")") as Cover, file.link as Name, year as Year,rating as Rating
from #Movie  
where !contains(file.folder, "88-Template")
sort rating desc

```
